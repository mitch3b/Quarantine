Video Chat

These are all free and have both mobile and pc support
  • Zoom - Works great but call starter needs to have a pro account
  • Skype - requires installation
  • Hangouts - requires gmail i think, UI is ok but a little annoying for groups of 4 or more
  • Discord - requires a little setup, although great to have the chat feature
  • Aws Chime - requires amazon.com account
  • Kast - targets viewing parties, etc
